Question 1: How do you specify a TCP socket in Python?
    By calling the socket.connect() method with a tuple parameter specifying the host (ip or hostname) and port number.

Question 2: What is the difference between a client socket and a server socket in Python?
    A client socket is created by calling the .connect(), .send(), and .recv() methods whereas a server si created by calling the .bind(), .listen(), and .accept() methods.
